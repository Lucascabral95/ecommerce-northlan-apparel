# Documentation

This directory holds project documentation for Northlane Apparel.

Current documents:

- `local-development.md`: local Docker Compose infrastructure and development URLs.
- `event-driven-flow.md`: RabbitMQ exchanges, retry/DLQ strategy and checkout saga flow.
- `payment-flow.md`: MOCK and Mercado Pago payment provider flow, webhook handling and local sandbox notes.
- `aws-architecture.md`: AWS ECS Fargate target architecture, HTTPS ALB design and cost notes.
- `deployment.md`: Terraform deployment workflow for the AWS dev environment.
- `i18n.md`: localized route and translation architecture.
- `observability.md`: Prometheus, Grafana, Loki and Alloy local observability stack.
- `testing-and-ci.md`: local quality gates, checkout/payment test coverage and GitHub Actions.
